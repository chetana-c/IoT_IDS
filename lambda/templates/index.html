Introduction:

In today's dynamic security landscape, the need for robust, intelligent, and efficient surveillance systems has become increasingly vital. Leveraging the power of Internet of Things (IoT) technology, this proposed project aims to develop an advanced surveillance system tailored for both home and enterprise use. The system utilizes motion detection through Passive Infrared (PIR) sensors, live video capture, and cutting-edge computer vision algorithms to effectively detect and respond to potential intrusions in real-time.

Objectives:

  1. Designing and implementing a reliable IoT framework for seamless integration of various surveillance components.

  2. Development of an intelligent motion detection system using PIR sensors for precise and accurate intrusion detection.

  3. Real-time video capturing and analysis through IoT-enabled cameras for effective monitoring of premises.

  4. Integration of computer vision algorithms to identify and distinguish between normal activities and potential security breaches.

  5. Deployment of a user-friendly interface for remote monitoring and control, enabling prompt actions in case of security threats.

  6. Ensuring data privacy and security through robust encryption protocols and secure communication channels.

Other methods of intrusion detection:

Looked into Wi-Fi enabled intrusion detection, where we can set up the raspberry pi to run a python script that scans for nearby Wi-Fi devices and checks their MAC addresses against a list of authorized devices(Can use the Scapy library to monitor Wi-Fi signals). Based on this, we can trigger the camera to capture visuals which will in turn trigger the detection algorithm. We can also run scheduled scans for nearby Wi-Fi devices based on requirement.
